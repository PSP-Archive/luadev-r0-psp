string ini.read ( string  path,
string  section,
string  key,
string  defecto 
)

Lee una entrada en un archivo .ini.

Si no encuentra la entrada, devuelve el argumento defecto

Parámetros:
path La ruta del archivo .ini
section La sección.
key El nombre de la clave.
defecto El valor por defecto si no se encuentra la clave en el archivo.
Devuelve:
El valor encontrado, o el valor por defecto.
