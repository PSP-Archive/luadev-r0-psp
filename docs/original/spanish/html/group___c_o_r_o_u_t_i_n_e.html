Resume o comienza a ejecutar una corutina.

Comienza a ejecutarse la función de la corutina, pasandole como argumentos, los argumentos extra dados al resume. Si la corutina ya está iniciada, los argumentos extra dados en el resume, serán devueltos como resultado a su coroutine.yield()

Parámetros:
corutina Corutina a resumir / empezar.
... Argumentos extra opcionales, para pasar coroutine.yield() o f.
Devuelve:
Un booleano + argumentos extra recibidos de coroutine.yield().
  • Si la corutina se ejecuta sin errores, devuelve true más todos los argumentos pasados a coroutine.yield.
  • Si la corutina tiene errores, devuelve false más el mensaje del error

string coroutine.status ( thread  corutina  ) 

Estado de una corutina.

Parámetros:
corutina Corutina a obtener el estado.
Devuelve:
Devuelve un string, con el estado de la corutina, éste puede ser:
  • "running" - Se está ejecutando ahora mismo.
  • "suspended" - Está en espera, o acaba de ser creada.
  • "normal" - Está activa, pero no ejecutandose (llamó a otra corutina)
  • "dead" - Ha acabado, o tiene algún error.

function coroutine.wrap ( function  f  ) 

Crea una nueva corutina, convertida a función.

Parámetros:
f Función a partir de la cual se creará la nueva corutina.
Devuelve:
Una función que, al ser llamada, hará un resume de la corutina. Cualquier argumento extra dado en esta función retornada serán pasados como argumentos extra en el resume. Devuelve lo mismo que resume, pero sin el primer booleano. En caso de error, el error se propaga.
