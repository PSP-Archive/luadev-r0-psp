Realiza una petición GET. Devolución en string.

Parámetros:
url Dirección del archivo a pedir. Ejemplos de urls:

  • Estilo "http://host.com/path/file.ext"
  • Estilo "http://host.com/path/"
  • Estilo "http://host.com"
size Tamaño en bytes a recibir. (OPCIONAL)
Devuelve:
Los datos descargados, en string.
nil http.get ( string  url,
string  file 
)

Realiza una petición GET. Guardar a archivo.

Usado con un parámetro string de segundo argumento, se ejecutará el diálogo por defecto de LuaDEV, con una barra de progreso, y descargará el archivo al lugar especificado por file .

Parámetros:
url Dirección del archivo a pedir. Ejemplos de urls:

  • Estilo "http://host.com/path/file.ext"
  • Estilo "http://host.com/path/"
  • Estilo "http://host.com"
file Ruta del archivo a guardar. Ejemplo "file.htm"
Devuelve:
Nada.
