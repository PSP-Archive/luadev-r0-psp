boolean usb.status (  ) 

Estado del driver del USB.

Estar activado implica que si se conecta el cable USB, se establecerá la conexión, pero no implica que esté realmente conectado.

Devuelve:
El estado del usb.
Excepciones:
- Aún devolviendo true, puede que el usb no esté conectado, véase usb::state().
table usb.state (  ) 

Determina el estado de la conexión con el USB.

Para saber si la conexión USB ha sido establecida se usará el campo "established".

Devuelve:
Una tabla con los siguientes campos: "activated", "connected" y "established". Todos de valor booleano.
