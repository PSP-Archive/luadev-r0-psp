Módulos

Lista de todos los módulos:
  • batt : Manejo de la batería.
  • cam : Funciones de la cámara (Go! Cam)
  • chipmunk : Motor físico bidimensional. (Fisicas 2D)
  • color : Creación y manejo de colores.
  • controls : Uso de los controles.
  • coroutine : Funciones multitarea.
  • draw : Dibujo de formas y líneas.
  • files : Manejo del sistema de archivos.
  • font : Fuentes
  • hprm : Manejo del hprm (remote)
  • http : Peticiones http.
  • image : Manejo de imágenes.
  • ini : Archivos de configuración.
  • io : Entrada / salida, manejo de archivos.
  • math : Funciones matemáticas.
  • os : Propiedades y funciones del sistema operativo.
  • power : Sistema de alimentación.
  • screen : Manejo de la pantalla.
  • sound : Objetos de sonido.
  • string : Funciones para el manejo de textos.
  • table : Manejo de tablas.
  • threadman : Multitarea por procesos.
  • timer : Temporizadores.
  • usb : Conexión USB.
  • wlan : Conexión a internet.
  • Sin módulo, funciones básicas.
