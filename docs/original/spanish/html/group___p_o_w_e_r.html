nil power.tick (  ) 

Evita que la PSP se ponga en "stand-by".

El tick es necesario enviarlo cada cierto tiempo, para que la PSP no vaya a "stand-by" automáticamente.
Además, si el usuario presiona cualquier tecla, el temporizador de "stand-by" vuelve a 0. (Igual que con esta función).

Devuelve:
Nada.
nil power.tick ( int  modo  ) 

Evita que la PSP se ponga en "stand-by".

El tick es necesario enviarlo cada cierto tiempo, para que la PSP no vaya a "stand-by" automáticamente.
Además, si el usuario presiona cualquier tecla, el temporizador de "stand-by" vuelve a 0. (Igual que con esta función).

Parámetros:
modo Uno de los siguientes:

  • power.tick.all - Evita que la PSP apague la pantalla y vaya a suspensión.
  • power.tick.display - Evita que la PSP apague la pantalla.
  • power.tick.suspend - Evita que la PSP vaya a suspensión.
